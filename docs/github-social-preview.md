# GitHub Social Preview

推荐上传文件：

- `assets/social/github-social-preview.png`

推荐规格：

- 尺寸：`1280 x 640`
- 格式：`PNG`
- 当前文件大小：远低于 `1 MB`

建议做法：

1. 打开 GitHub 仓库 `amazon-skills`
2. 进入 `Settings`
3. 找到 `Social preview`
4. 上传 `assets/social/github-social-preview.png`

仓库首页 README 顶部展示图使用的是：

- `assets/social/github-cover.svg`

说明：

- `github-cover.svg` 适合在 README 中展示
- `github-social-preview.png` 适合上传到 GitHub 的 Social preview
