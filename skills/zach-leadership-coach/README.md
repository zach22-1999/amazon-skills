## 这个 Skill 做什么

亚马逊的 16 条领导力原则是全球最被广泛借鉴的管理框架之一。但大多数人只在面试时才翻出来看看，日常工作中用不起来。

`zach-leadership-coach` 把这套原则变成了一个**可交互的个人教练**——你描述你遇到的情况，它用最相关的原则帮你理清思路、做出更好的决定。

不是照本宣科念原则，而是像一个懂这套体系的导师在跟你聊天。

---

## 六种工作模式

| 你在说什么 | Skill 做什么 | 输出风格 |
|---|---|---|
| 遇到一个难题不知道怎么办 | **问题梳理** — 用原则引导你思考 | 教练式对话 |
| 想检查自己一段时间的表现 | **定期自检** — 逐条打分 + 30天改进计划 | 框架式评估 |
| 在几个选项间犹豫不决 | **决策评估** — 多原则维度对比分析 | 对比表格 + 建议 |
| 刚做完一件事想复盘 | **复盘反思** — 提炼具体教训 | 时间线 + 教训清单 |
| 想规划下一阶段的方向 | **目标规划** — 用原则校准方向 | 目标拆解 + 行动计划 |
| 和人打交道遇到困难 | **人际沟通** — 角色互换 + 沟通建议 | 教练式引导 |

Skill 会根据你说的话自动判断用哪个模式，不需要你指定。

---

## 两个设计亮点

### 1. 信息不足时先提问

如果你说的太模糊（比如"我想跳槽"），Skill 不会硬分析，而是先问你 2-3 个针对性问题把关键信息补齐。基于模糊信息的分析看起来完整，实际上没用。

### 2. 双格式输出

分析完成后同时生成两个文件：
- **Markdown 文件** — 带 YAML 元数据头，方便归档到个人知识库
- **HTML 可视化** — 雷达图、进度条、对比卡片，浏览器直接打开

## 安装

### 方式 1：Claude Code 命令安装

```bash
claude install-skill https://github.com/zach22-1999/amazon-skills/tree/main/skills/zach-leadership-coach
```

### 方式 2：手动安装

```bash
git clone https://github.com/zach22-1999/amazon-skills.git
cd amazon-skills

# 复制到你的工作目录
TARGET=~/your-workspace
mkdir -p "$TARGET/.claude/commands"
cp -r skills/zach-leadership-coach "$TARGET/.claude/commands/"
```

## 使用方法

### 自然语言（直接描述你的情况）

```
我团队里有个人能力很强但总是不配合协作，其他人都不敢发言了，怎么处理？

我做运营半年了，想用领导力原则给自己做个全面体检

现在有两个选择：继续在公司做主管，还是跳槽去创业公司带团队，帮我分析一下

刚做完一个产品上线项目，结果不太理想，想复盘一下
```

### Skill 命令

```
/zach-leadership-coach
```

然后描述你的情况即可。

## 适合谁用

- **不只是亚马逊员工** — 这套原则适用于任何需要做决策、带团队、规划方向的人
- **管理者** — 团队管理、人际沟通、绩效评估
- **个人成长** — 定期自检、目标规划、职业决策
- **创业者** — 方向选择、合伙人沟通、战略规划

---

## 包含的 16 条原则

顾客至尚 · 主人翁精神 · 创新简化 · 决策正确 · 好奇求知 · 选贤育能 · 最高标准 · 远见卓识 · 崇尚行动 · 勤俭节约 · 赢得信任 · 刨根问底 · 敢于谏言服从大局 · 达成业绩 · 成为最佳雇主 · 成功和规模带来更大责任

每条原则都包含官方原文、核心要义解读、日常应用场景，以及原则之间的张力关系分析。
